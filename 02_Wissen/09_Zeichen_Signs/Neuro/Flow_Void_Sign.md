# Flow Void Sign

## Definition
Signalverlust (schwarz) im Aquaeductus mesencephali in T2-gewichteten MRT-Bildern durch turbulente Liquorrömung. Ein *verstärkter* Flow Void ist typisch für den Normaldruckhydrozephalus (NPH).
