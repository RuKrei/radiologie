# Pankreastumoren

- **Duktales Adenokarzinom:** Macht den Ductus Wirsungianus und D. choledochus dicht ("Double Duct Sign"). Hypodens im CT. Miese Prognose. [Radiopaedia](https://radiopaedia.org/articles/pancreatic-ductal-adenocarcinoma)
- **Zystische Läsionen (Der goldene Merkspruch):**
  - *Seröses Zystadenom:* "Grandma" (alt), Mikrozysten (Schwammartig), zentrale Narbe/Verkalkung. Gutartig.
  - *Muzinöses Zystadenom:* "Mother" (Mittelalt), Makrozysten, Korpus/Schwanz. Prämaligne!
  - *IPMN:* "Daughter". Kommuniziert mit dem Gangsystem! Main-Duct ist böse (Operation), Branch-Duct wird beobachtet.
