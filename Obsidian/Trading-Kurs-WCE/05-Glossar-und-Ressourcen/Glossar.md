# Glossar & Abkürzungen

Hier findest du die wichtigsten Begriffe aus dem World Class Edge (WCE) Universum und dem professionellen Orderflow-Trading.

## Akronyme der Marktstruktur

| Akronym | Vollständiger Name | Erklärung |
| :--- | :--- | :--- |
| **AMT** | **A**uction **M**arket **T**heory | Theorie, die den Markt als Auktionsprozess beschreibt (Suchen nach Fair Value). |
| **POC** | **P**oint **o**f **C**ontrol | Der Preis mit dem höchsten gehandelten Volumen im Profil. |
| **VA** | **V**alue **A**rea | Der Preisbereich, in dem ca. 70% des Volumens gehandelt wurde. |
| **VAH** | **V**alue **A**rea **H**igh | Die obere Grenze der Value Area. |
| **VAL** | **V**alue **A**rea **L**ow | Die untere Grenze der Value Area. |
| **HVN** | **H**igh **V**olume **N**ode | Ein Preisbereich mit viel Volumen ("Berg" im Profil). Wirkt anziehend/bremsend. |
| **LVN** | **L**ow **V**olume **N**ode | Ein Preisbereich mit wenig Volumen ("Tal" im Profil). Preis bewegt sich hier oft schnell durch. |
| **TPO** | **T**ime **P**rice **O**pportunity | Auch "Market Profile". Ähnlich wie Volumen Profil, misst aber, wie *lange* der Preis auf einem Niveau war, nicht wie viel gehandelt wurde. |

## Orderflow Begriffe

| Begriff | Erklärung |
| :--- | :--- |
| **Aggressor** | Der Marktteilnehmer, der die Initiative ergreift (Market Order). Er "nimmt" Liquidität aus dem Buch. |
| **Passive Order** | Eine Limit Order, die im Orderbuch wartet (Liquidität bereitstellt). |
| **Delta** | Die Differenz zwischen aggressivem Kaufvolumen und aggressivem Verkaufsvolumen. (Positives Delta = mehr aggressive Käufer). |
| **CVD** | **C**umulative **V**olume **D**elta. Die Summe des Deltas über einen Zeitraum. Zeigt, wer langfristig "drückt". |
| **Absorption** | Wenn aggressive Käufer kaufen, der Preis aber nicht steigt (weil ein großer passiver Verkäufer alles aufsaugt/absorbiert). Ein starkes Umkehrsignal. |
| **Imbalance** | Im Footprint-Chart: Wenn auf einer Seite deutlich mehr Volumen gehandelt wurde als auf der Gegenseite (oft 300%+ mehr). Zeigt Aggressivität. |
| **DOM** | **D**epth **o**f **M**arket. Das Orderbuch. Zeigt alle wartenden Limit-Orders. |

## Allgemeine Trading Begriffe

| Begriff | Erklärung |
| :--- | :--- |
| **Edge** | Der statistische Vorteil. Eine Strategie, die über viele Trades hinweg profitabel ist. |
| **R:R (CRV)** | **R**isk to **R**eward (Chancen-Risiko-Verhältnis). Wie viel riskiere ich (bis Stop Loss) vs. wie viel kann ich gewinnen (Take Profit). |
| **Stop Loss (SL)** | Eine Order, die den Trade automatisch schließt, wenn er gegen dich läuft, um Verluste zu begrenzen. |
| **Take Profit (TP)** | Eine Order, die Gewinne automatisch mitnimmt. |
| **Long** | Auf steigende Preise setzen (Kaufen). |
| **Short** | Auf fallende Preise setzen (Verkaufen). |
| **Range** | Seitwärtsbewegung des Marktes (Balance). |
| **Trend** | Gerichtete Bewegung des Marktes (Imbalance). |
